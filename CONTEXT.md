# LifeOS

LifeOS is personal AI assistant software centered on a user's ongoing chats with an AI counterpart.

## Language

**Chat**:
A durable conversation thread between the user and the AI counterpart that can be continued over time.
_Avoid_: Conversation, session, visit, browser session

**Agent**:
The product-facing name for the AI counterpart in the chat interface.
_Avoid_: Assistant

**Prompt**:
The ordered chat history and current input supplied to the AI model.
_Avoid_: Context, transcript

**Message**:
An item in a Chat, including conversational text and non-text events such as tool calls, attachments, errors, or system notices.
_Avoid_: Conversation event, timeline item
